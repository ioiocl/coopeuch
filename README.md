# Coopeuch

Java 1.8, Node 12.x
Bases de datos : H2

# Para instalar y correr el Backend correr los siguientes comandos

mvn install
cd target
java -jar rest-demo-0.0.1-SNAPSHOT.jar


# Para instalar el Front  ir a la carpeta front-end y ejecutar los siguientes comandos
```
> npm install
> npm start
```
